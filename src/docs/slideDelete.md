# slide-delete
该插件实现了滑动删除的功能
#### Install with npm: 
    npm install
#### start webpack-dev-server
    npm run start:dev
#### 打包
    npm run build
#### 效果
![image](https://github.com/KuangPF/Web-Plugin/blob/master/src/docs/assets/demoImg/slide-delete.gif)

参考了项目:https://github.com/DeronEndless/vue-left-slip-delete, 在该项目基础上做了修改.
